Java Bus Reservation System

A simple Java-based console application that simulates a bus reservation system. This project demonstrates core OOP concepts such as classes, objects, inheritance, and basic file handling. Users can book, cancel, and view seat availability.

Key Features

  • Interactive menu for booking and canceling tickets
  • Displays available and reserved seats
  • Object-Oriented approach using Java classes
  • Handles input validation and duplicate bookings

Technologies Used

  • Core Java
  • OOP Concepts (Encapsulation, Inheritance)
  • Scanner for user input
