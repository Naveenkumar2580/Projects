Java Expense Tracker

This console-based Java application helps users manage and track their daily expenses. It includes features for adding, viewing, categorizing, and summarizing expenses, making it an ideal project for learning object-oriented programming and file handling.

Key Features

  • Add expenses with category and date
  • View all expenses or filter by category/date
  • Summarize total and category-wise expenses
  • Data persistence using file I/O
  • User-friendly console interface

Technologies Used

  • Java
  • OOP (Classes, Objects)
  • File I/O for persistent storage
  • Scanner for user input
