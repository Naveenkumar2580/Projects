Java Console Calculator

A basic calculator application developed using Java for console environments. This project allows users to perform simple arithmetic operations such as addition, subtraction, multiplication, and division. It emphasizes the core programming logic and control flow structures essential for beginners in Java development.

Key Features

  • Interactive console-based menu
  • Supports +, -, *, / operations
  • Handles invalid inputs and divide-by-zero errors gracefully
  • Uses loops, conditionals, and user input with Scanner

Technologies Used

  • Java
  • Scanner for user input
  • Switch-case statements
