Java Contact Book

This is a console-based Java application that functions as a simple contact management system. Users can add, view, update, delete, and search contacts through an interactive text interface. It's a great way to learn object-oriented concepts and file handling in Java.

Key Features

  • Add and manage multiple contacts
  • Search contacts by name or phone number
  • Edit or delete saved contacts
  • Persistent storage using file I/O
  • Simple and beginner-friendly UI through console

Technologies Used

  • Java
  • OOP (Classes, Objects)
  • File I/O (for storing contact data)
  • Scanner for user input
